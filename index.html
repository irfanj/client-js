JavaScript Client Examples

This page shows you the examples for the JavaScript client library.

There are two types - those which have HTML pages for you to see the results, and those which are just code snippets with console logging.

If you're unsure where to start, checkout this example.

Quick start

Interactive examples

  • Signed auth - using ID / API key authentication.
  • Custom signed auth - using a custom signing function for authentication.
  • Signing server - using a signing server for authentication.

Code examples

You need to download and run these yourself, as you need to set up their options in code.

  • Callback function - using a single function for callback.
  • Callback chaining - how to chain callbacks using jQuery Promises.
  • Callback object - specifying callbacks with an object.
  • Callback objects and chaining - using both above techniques simultaneously.
  • Configuration - example of providing configuration to the library.
  • Query chaining - chaining multiple queries together.

Server examples

Examples for server components you can run yourself.

  • Server signer - sign user's queries on your server.
  • Custom sign server - validate and sign user's queries.
